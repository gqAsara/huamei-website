# Pinyin reference

Canonical romanization for brand and place names used across
huamei.io. Translator must consult this file before writing any
pinyin and add new entries (with human approval) when names appear
that aren't yet here.

| Hanzi | Pinyin (Hanyu) | Notes |
|-------|----------------|-------|
| 華美 | Huamei | Brand. No tone marks in URLs or marketing copy. |
| 河南 | Henan | Factory province. |
| 浙江 | Zhejiang | Factory province. |
| 四川 | Sichuan | Factory province. |
| 贵州 | Guizhou | Factory province. |

## Rules

- Hanyu Pinyin only. Never Wade-Giles (Hua-mei → wrong).
- No tone marks in body copy or URLs. Tone marks acceptable in
  pronunciation guides only.
- For traditional Chinese characters that are common in luxury
  positioning (華美 vs 华美), the brand uses traditional 華美 in
  wordmark, simplified 华美 in body copy. Document both forms when
  adding new entries.
